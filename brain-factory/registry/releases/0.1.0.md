# Framework release 0.1.0

- Date: 2026-06-17
- Manifest compatibility: `>= 1.0.0`

## Summary

Initial brain-factory foundation. Establishes the hub-and-spoke brain model, the
core/extension boundary, the manifest contract, the registry and improvement
loop, the inspect-first onboarding engine spec, and the cross-platform adapter
seam.

## Core modules (all `added`)

| Module | Change | Notes |
| --- | --- | --- |
| `operating-contract` | added | Templated single-law contract with thin per-repo pointers. |
| `session-ritual-hooks` | added | SessionStart/SessionEnd continuity hooks (cross-platform). |
| `continuity-log` | added | `05-logs/` continuity record + master index. |
| `capabilities-map` | added | `01-docs/CAPABILITIES.md` generated from code. |
| `docs-mesh` | added | Link/diagram/freshness anti-drift checks. |

| `core-commands` | added | The core command set (see core-commands/CATALOG.md). |

Nothing — there are no prior versions to upgrade from. New brains are provisioned
at this version.
