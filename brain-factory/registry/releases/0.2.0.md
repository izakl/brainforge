# Framework release 0.2.0

- Date: 2026-06-24
- Manifest compatibility: `>= 1.0.0`

## Summary

A capability wave on the Phase 1 foundation: the down-sync now executes
end-to-end, the engine installs as a CLI (pipx/npx), and releases are automated
behind a version-parity guardrail. See the [CHANGELOG](../../../CHANGELOG.md)
for the full list.

## Action for brains

Run `<prefix>-upgrade` (or `run.sh upgrade-brain --brain <brain>`): dry-run to
preview, then `--apply`. It refreshes the enabled core modules to 0.2.0, bumps
`framework_version` and `synced_from`, and preserves extensions. Pre-existing
(`adopted: false`) modules are surfaced for manual review.
