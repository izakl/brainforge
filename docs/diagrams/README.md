# Diagrams

Framework diagrams are authored inline as Mermaid in each doc's `## Diagram`
section. GitHub renders Mermaid natively in Markdown, and the documentation site
renders it live (Material's `pymdownx.superfences`), so no separate image files
are needed.

The earlier static **SVG companions were retired** — see
[ADR 0024](../adr/0024-retire-svg-companions.md). The `## Diagram` Mermaid
convention itself remains: see
[ADR 0009](../adr/0009-mermaid-as-primary-diagram-format.md) and
[ADR 0010](../adr/0010-diagrams-convention.md).
