# Diagrams (SVG companions)

This directory stores SVG companions for Mermaid diagrams in framework docs.

Convention:

- Each SVG mirrors a source doc's `## Diagram` Mermaid block.
- Filename matches the source doc slug (source filename without `.md`).
- To regenerate: render the Mermaid block from the source doc using your preferred Mermaid → SVG tool; verify nodes/edges match before committing.
